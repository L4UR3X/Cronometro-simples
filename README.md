# Cronômetro Simples

Projeto de **cronômetro simples** criado com **HTML, CSS e JavaScript**.

---

## Sobre

Este repositório contém uma página web que funciona como um cronômetro. O código é simples e ideal para treinar lógica de programação front-end.

---

## Estrutura

Arquivos principais:
- `index.html` — estrutura da página
- `style.css` — estilos visuais
- `script.js` — lógica do cronômetro

---

## Tecnologias

- HTML
- CSS
- JavaScript
